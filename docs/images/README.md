# README photos

Drop your photos here with these exact filenames — the main README embeds them:

| Filename | What to photograph |
|----------|--------------------|
| `overview.jpg` | Hero shot — the gate + a node, or a wide install shot |
| `xiao-esp32s3.jpg` | A Seeed XIAO ESP32-S3 (ideally with its external antenna) |
| `moonshan-ms-go-1.jpg` | The Moonshan MS-GO-1 gate opener |
| `power-converter.jpg` | The 24 V → 5 V USB-C buck converter |
| `gate-wiring.jpg` | The gate node wired to the opener + power converter |

JPG or PNG both work (update the extension in `README.md` if you use PNG).
Until you add them, the README image links will show as broken — that's expected.
